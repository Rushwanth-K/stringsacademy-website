# 🎸 StringsAcademy – Guitar Learning Website

A modern and responsive landing page for a guitar academy, built using HTML, CSS, and JavaScript.

## 🚀 Features

* Responsive design (mobile-friendly)
* Smooth scrolling navigation
* Multi-section landing page (Hero, Services, About, Contact)
* Separate enrollment page with form validation
* Clean UI using reusable CSS variables

## 🛠️ Tech Stack

* HTML5
* CSS3 (Flexbox + Grid)
* JavaScript (Vanilla)

## 📂 Pages

* `index.html` – Main landing page
* `enroll.html` – Enrollment form

## 📌 What I Learned

* Building a complete website structure
* Form handling using JavaScript
* Responsive layouts using CSS Grid & Flexbox
* UI/UX design basics.
